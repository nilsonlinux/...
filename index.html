Nilsonlinux. Todos direitos reservados. 2018

Nilsonlinux. Sábado 21/07/2018

O ChromeOS foi a resposta da Google a outras empresas que insistem em ter sistemas operativos fechados e que o utilizador controla apenas em parte. Assente no Chrome, este SO não teve o crescimento esperado, sendo sempre associado a máquinas com recursos limitados e que existisse uma forma simples de ser instalado. Mas esses tempos estão ultrapassados e surge agora o CloudReady, uma forma simples de instalar o ChromeOS. Tudo está pronto a ser usado, com um processo simples de criação da necessária pen USB de instalação, de forma fácil e perfeitamente compreensível. Porque normalmente o suporte para hardware é o maior problema encontrado e por essa razão as dificuldades surgem, os criadores do CloudReady tiveram o cuidado de o preparar para correr numa longa lista de hardware. Essa lista pode ser consultada neste documento. Para apoiar neste processo, que como dissemos antes é extremamente simples, a Neverware criou um conjunto de instruções, que podem ser acompanhadas e seguidas.

Como instalar o CloudReady?

Para criar uma pen de instalação do CloudReady só precisa realizar os 3 passos que apresentamos abaixo. Depois disso é fazer uma normal instalação, arrancando de uma pen USB. Instale no seu Chrome o plugin Chrome Recovery Tool a partir da Chrome Web Store. Descarregue a versão Individual do CloudReady a partir da Neverware. Note que existem outras versões (educação e enterprise) mas estas são pagas. Lance a ferramenta de recuperação e siga as instruções, indicando o local para onde descarregou a imagem do CloudReady e também onde está a pen. Depois de ter a sua pen criada, só precisa de reiniciar a máquina onde quer instalar o CloudReady. Tenha em atenção que deve ter a pen colocada no PC e que a BIOS deve estar configurada para arrancar dessa mesma drive. O CloudReady permite que seja executado em modo Live, em que não é instalado no PC, ou através de uma instalação completa, ficando o ChromeOS instalado de forma definitiva. Notem que esta última opção elimina todos os dados do PC e que por isso devem realizar uma cópia de segurança antes.
